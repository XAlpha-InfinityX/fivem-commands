Commands list

  • Basic keybinds

    • • M - vMenu menu (opens vMenu)
    • • L - Hide/show chat (hide or show the chat menu)

  • FivePD keybinds

    • • F11 - Duty menu (menu to go on and off duty)
    • • Y - Accept callouts (key to accept callouts from dispatch)
    • • Z - Callout menu (menu to either request backup from your department, or call a scene/callout as Code 4 or completed)
    • • G - Scene menu (menu to place objects such as traffic cones or set a speed zone for an area)
    • • X - Ped interaction menu (menu to interact with pedestrians)
    • • 0 (zero) - Waypoint to closest jail (set waypoint to closest jail when arrested ped in vehicle)
    • • E - Place cuffed ped in vehicle (when nearby cuffed ped and vehicle, places them in vehicle)
    • • Hold E (around 2 sec.) - Stop ped/cuff ped (stops a ped or cuffs a stopped ped)
    • • LShift + E - Spike strips (when standing by trunk of applicable police vehicle, places spike trips, or picks them up if already placed)
    • • U - Ped in jail (when near an arrested ped at a jail drop off point, puts them in jail)
    • • J - ID card panel (after receiving a ped's license, opens the ID card panel)

  • Vehicle keybinds

    • • Q - Toggle police lights (when in police cruiser, toggles lights on and off)
    • • B - Open MDT (key to open MDT while in your vehicle in order to search peds/license plates and write citations/arrest reports)
    • • LShift - Traffic stop (positioning your police cruiser behind a vehicle and pressing this key will mark it on the minimap, once marked you can use your lights to signal for it to pull over)
    • • F9 - Toggle ALPR (while in police cruiser, this key automatic license plate reader will notify you of any vehicles that have flags on their license plate)
    • • X - Traffic stop positioning menu (after pulling over vehicle, opens traffic marker controls to move vehicle)
    • • 1, 2, 3 - Siren wails (toggles between primary, secondary, and tertiary siren when police cruiser lights are on)
    • • O (letter) - Luxart menu (opens the Luxart menu)
    • • LCTRL - Enable/disable cruise control (after enabling cruise control in the duty menu, this enables or disables cruise control)
    • • NUM + - Increase cruise control speed (increases the cruise control speed)
    • • NUM - - Decrease cruise control speed (decreases the cruise control speed)

  • AI backup keybinds

    • • F10 - AIBackup menu (menu to go on duty and request backup)
    • • NUM + - Call backup units (key to call backup units when on duty in AIBackup menu)
    • • NUM - - Dismiss backup units (key to dismiss AI backup once all units are on scene)
    • • PageDown + aiming at Ped - Backup units stop aiming (this combination will make backup units on scene stop aiming)
    • • PageUp - Backup exits vehicle (key to make backup units in vehicle stop their car, exit vehicle, and come to your position when key pressed)
    • • Del - Dismiss backup units (all backup units are dismissed)
    • • /getout i - Get unit out of vehicle (command to get unit 'i' out of vehicle, i is a number between 1 and max number of backup units, useful if they get stuck)
    • • /getin i - Get unit into their vehicle (command to get unit 'i' into their vehicle, i is a number between 1 and max number of backup units, useful if they get stuck)
    • • /onduty - Change duty status (command to change status to "on duty")
    • • /offduty - Change duty status (command to change status to "off duty")
    • • /stopNotify - Clear notifications (command to clear top right notifications)

  • Chat keybinds

    • • /eup - Emergency Uniforms Pack menu (opens EUP menu to allow you to customize gear and outfits)

  • Terminal keybinds

    • • quit - Quit FiveM (quits the game)
    • • disconnect - Return to FiveM main menu (disconnects from server and returns to main menu)
    • • restart <resource_name> - Restart resource (restarts the specified resource)
    • • refresh - Refreshes all resources (refreshes and loads all resource changes)
